# 面向对象

标签（空格分隔）： 面向对象

---

##**面向对象Object Oriented**
- 面向对象是一种对现实世界理解和抽象的方法，是计算机编程技术发展到一定阶段后的产物。
- 面向对象编程（Object Oriented Programming，OOP）是一种编程方式和思想。
	- 在面向对象语言中，理解类和对象的概念

	- **三大特性**：封装 继承 多态

	- 类描述了一组有相同特性（属性）和相同行为（方法）的对象
		- 属性：描述对象的特征
		- 方法：对象具备的功能

	- 对象是类的具体实现
###**工厂方法**
- ES5.1没有类的概念，是不用关键字class关键类
- 用函数模拟类

```
//普通的创建对象的方法
let user1 = new Object();  // {}

user1.name = 'wang';
user1.age = 30;
user1.coding = function (){
	console.log('会写代码');	
}

let user2 = new Object();
user2.name = 'Reo';
user2.age = 28;
user1.coding = function (){
	console.log('会写代码');
}

```

使用工厂方法创建对象
```
// 工厂方法
function createUser(name,age){
	let user1 = new Object(); 
	user1.name = name;
	user1.age = age;
	user1.coding = function (){
		console.log('会写JS代码');	
	}
	return user1;
}

let user3 = createUser('mmo',30)
let user4 = createUser('mmo',30)

console.log(user3);
console.log(user4);
```
